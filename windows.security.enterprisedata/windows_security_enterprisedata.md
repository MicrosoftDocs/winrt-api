# Windows.Security.EnterpriseData

## -description

Contains classes that support Windows Information Protection (WIP). For the full developer picture of how WIP relates to files, buffers, the clipboard, networking, background tasks, and data protection under lock, see [Windows Information Protection](https://docs.microsoft.com/windows/uwp/enterprise/wip-hub).

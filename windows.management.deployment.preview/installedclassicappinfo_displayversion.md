# Windows.Management.Deployment.Preview.InstalledClassicAppInfo.DisplayVersion

## -description
Gets the display version for an installed classic desktop app.

## -property-value
The version of the app as a [String](/dotnet/api/system.string?view=dotnet-uwp-10.0&preserve-view=true) that can be displayed.

# Windows.Web.Http.Headers.HttpNameValueHeaderValue.Parse

## -description
Converts a string to an [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) instance.

## -parameters
### -param input
A string that represents the name and value.

## -returns
An [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) instance.

## -exceptions
### E_INVALIDARG

The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not valid name and value information.

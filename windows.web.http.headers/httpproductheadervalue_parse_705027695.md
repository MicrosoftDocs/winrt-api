# Windows.Web.Http.Headers.HttpProductHeaderValue.Parse

## -description
Converts a string to an [HttpProductHeaderValue](httpproductheadervalue.md) instance.

## -parameters
### -param input
A string that represents the product name and version.

## -returns
An [HttpProductHeaderValue](httpproductheadervalue.md) instance.

## -exceptions
### E_INVALIDARG

The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not valid product name and version information.

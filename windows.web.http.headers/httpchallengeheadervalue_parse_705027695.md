# Windows.Web.Http.Headers.HttpChallengeHeaderValue.Parse

## -description
Converts a string to an [HttpChallengeHeaderValue](httpchallengeheadervalue.md) instance.

## -parameters
### -param input
A string that represents authentication header value information.

## -returns
An [HttpChallengeHeaderValue](httpchallengeheadervalue.md) instance.

## -exceptions
### E_INVALIDARG

The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not valid authentication header information.

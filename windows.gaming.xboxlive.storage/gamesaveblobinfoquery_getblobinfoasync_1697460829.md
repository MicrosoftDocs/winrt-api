# Windows.Gaming.XboxLive.Storage.GameSaveBlobInfoQuery.GetBlobInfoAsync

Type: **Number** \[JavaScript\] | [System.UInt32](https://docs.microsoft.com/dotnet/api/system.uint32?redirectedfrom=MSDN) \[.NET\] | **uint32** \[C++\]

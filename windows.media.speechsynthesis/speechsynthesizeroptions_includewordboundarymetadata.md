# Windows.Media.SpeechSynthesis.SpeechSynthesizerOptions.IncludeWordBoundaryMetadata

## -description
Gets or sets whether word boundary metadata is added to a [SpeechSynthesisStream](speechsynthesisstream.md) object.

## -property-value
**true** if word boundaries are marked. Otherwise, **false**.

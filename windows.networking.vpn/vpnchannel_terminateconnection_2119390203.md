# Windows.Networking.Vpn.VpnChannel.TerminateConnection

## -description
Allows a VPN plugin to indicate an error message and terminate the connection attempt performed inside the [IVpnPlugin.Connect](ivpnplugin_connect.md) method. Should not be called on reconnect cases.

## -parameters
### -param message
The text of the error message.

# VpnPacketBufferStatus

## -description
Defines constants that specify the possible states of the VPN packet buffer.

## -enum-fields
### -field Ok:0
Specifies that the state is ok.

### -field InvalidBufferSize:1
Specifies an invalid buffer size.

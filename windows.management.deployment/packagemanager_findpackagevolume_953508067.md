# Windows.Management.Deployment.PackageManager.FindPackageVolume

## -description
Gets the specified volume.

## -parameters
### -param volumeName
The volume media ID, which is a [GUID](/windows/win32/api/guiddef/ns-guiddef-guid) value.

## -returns

# Windows.Foundation.Diagnostics.LoggingFields.AddGuidArray

## -description
Adds a [GUID](/windows/win32/api/guiddef/ns-guiddef-guid) array field with the specified field name.

## -parameters
### -param name
Name of the event field.

### -param value

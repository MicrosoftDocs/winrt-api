# Windows.UI.Input.GestureRecognizer.MouseWheelParameters

## -description
Gets a set of properties that are associated with the wheel button of a mouse device.

## -property-value
The collection of wheel button properties. The system defaults should be checked to ensure the best user experience for your app.

## -remarks
The properties exposed through [MouseWheelParameters](gesturerecognizer_mousewheelparameters.md) are used in conjunction with the **Wheel** settings in the **Mouse** control panel to define the UI experience when using the wheel button.

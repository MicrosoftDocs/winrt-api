# Windows.Media.Capture.CapturedPhoto

## -description
Represents a photo taken from a capture device such as a webcam attached.

## -remarks
Get an instance of this class by calling [LowLagPhotoCapture.CaptureAsync](lowlagphotocapture_captureasync_2078018041.md). For how-to guidance for using **LowLagPhotoCapture**, see [Basic photo, video, and audio capture with MediaCapture](/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture).

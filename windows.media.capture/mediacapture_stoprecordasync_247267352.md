# Windows.Media.Capture.MediaCapture.StopRecordAsync

## -description
Stops recording.

## -returns
Returns a [IAsyncAction](../windows.foundation/iasyncaction.md) object that is used to control the asynchronous operation.

## -remarks
Before playing back a video, the [RandomAccessStream](../windows.storage.streams/randomaccessstream.md) should be closed when [StopRecordAsync](mediacapture_stoprecordasync_247267352.md) completes.

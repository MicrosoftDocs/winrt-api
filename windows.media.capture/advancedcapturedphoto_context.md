# Windows.Media.Capture.AdvancedCapturedPhoto.Context

## -description
Gets the app-defined context object associated with the advanced photo capture operation, if one was provided in the call to [AdvancedPhotoCapture.CaptureAsync(Object)](advancedphotocapture_captureasync_651770772.md).

## -property-value
The app-defined context object associated with the advanced photo capture operation.

## -remarks
One overload of [AdvancedPhotoCapture.CaptureAsync](advancedphotocapture_captureasync.md) allows you to provide an app-defined object that conveys context information.

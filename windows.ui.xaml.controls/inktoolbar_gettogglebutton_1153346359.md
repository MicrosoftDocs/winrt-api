# Windows.UI.Xaml.Controls.InkToolbar.GetToggleButton

## -description
Retrieves a reference to an [InkToolbarToggleButton](inktoolbartogglebutton.md) object from the toggle button group of the [InkToolbar](inktoolbar.md).

## -parameters
### -param tool
Indicates the type of [InkToolbarToggleButton](inktoolbartogglebutton.md) to retrieve.

## -returns
The [InkToolbarToggleButton](inktoolbartogglebutton.md) specified.

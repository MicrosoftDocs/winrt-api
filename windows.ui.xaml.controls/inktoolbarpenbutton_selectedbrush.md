# Windows.UI.Xaml.Controls.InkToolbarPenButton.SelectedBrush

## -description
Gets the brush used to draw the ink stroke.

## -property-value
The brush used to draw the ink stroke. The default value is 0 ([Black](../windows.ui/colors_black.md)).

# Windows.UI.Xaml.Controls.TextBox.BeforeTextChanging

## -description

Occurs synchronously when the text in the text box starts to change, but before the **Text** property is updated.



## -xaml-syntax

```xaml
<TextBox BeforeTextChanging="eventhandler"/>
```

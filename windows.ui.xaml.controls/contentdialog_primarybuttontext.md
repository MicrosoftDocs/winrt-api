# Windows.UI.Xaml.Controls.ContentDialog.PrimaryButtonText

## -description
Gets or sets the text to display on the primary button.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.PrimaryButtonText](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.primarybuttontext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<ContentDialogÂ PrimaryButtonText="string"/>
```


## -property-value
The text to display on the primary button. To hide this button, set the text to **null** or string.empty. The default is empty.

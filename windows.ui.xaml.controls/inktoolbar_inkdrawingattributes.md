# Windows.UI.Xaml.Controls.InkToolbar.InkDrawingAttributes

## -description
Gets the [InkStroke](../windows.ui.input.inking/inkstroke.md) properties specified through the [InkToolbar](inktoolbar.md).



## -property-value
The [InkStroke](../windows.ui.input.inking/inkstroke.md) properties.

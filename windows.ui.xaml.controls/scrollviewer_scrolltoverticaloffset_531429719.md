# Windows.UI.Xaml.Controls.ScrollViewer.ScrollToVerticalOffset

## -description
> [!NOTE]
> ScrollToVerticalOffset may be altered or unavailable for releases after WindowsÂ 8.1. Instead, use [ChangeView](scrollviewer_changeview_1425504772.md).

Scrolls the content that is within the [ScrollViewer](scrollviewer.md) to the specified vertical offset position.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.ScrollToVerticalOffset](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.scrolltoverticaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param offset
The position that the content scrolls to.

# Windows.UI.Xaml.Controls.PasswordBox.PasswordChanging

## -description

Occurs synchronously when the text in the password box starts to change, but before it is rendered.



## -xaml-syntax

```xaml
<PasswordBox PasswordChanging="eventhandler"/>
```

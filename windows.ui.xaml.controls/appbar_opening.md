# Windows.UI.Xaml.Controls.AppBar.Opening

## -description
Occurs when the [AppBar](appbar.md) starts to change from hidden to visible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBar.Opening](/windows/winui/api/microsoft.ui.xaml.controls.appbar.opening) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<AppBar Opening="eventhandler"/>
```

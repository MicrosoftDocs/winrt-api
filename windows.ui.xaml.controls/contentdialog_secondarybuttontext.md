# Windows.UI.Xaml.Controls.ContentDialog.SecondaryButtonText

## -description
Gets or sets the text to be displayed on the secondary button.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.SecondaryButtonText](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.secondarybuttontext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<ContentDialogÂ SecondaryButtonText="string"/>
```


## -property-value
The text to be displayed on the secondary button. To hide this button, set the value to **null** or **string.empty**.

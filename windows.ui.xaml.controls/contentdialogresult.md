# ContentDialogResult

## -description
Specifies identifiers to indicate the return value of a [ContentDialog](contentdialog.md)



## -enum-fields
### -field None:0
No button was tapped.

### -field Primary:1
The primary button was tapped by the user.

### -field Secondary:2
The secondary button was tapped by the user.

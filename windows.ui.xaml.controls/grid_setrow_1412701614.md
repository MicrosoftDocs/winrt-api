# Windows.UI.Xaml.Controls.Grid.SetRow

## -description
Sets the value of the [Grid.Row](/uwp/api/windows.ui.xaml.controls.grid#xaml-attached-properties) XAML attached property on the specified [FrameworkElement](../windows.ui.xaml/frameworkelement.md).

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.Grid.SetRow](/windows/winui/api/microsoft.ui.xaml.controls.grid.setrow).

## -parameters
### -param element
The target element on which to set the [Grid.Row](/uwp/api/windows.ui.xaml.controls.grid#xaml-attached-properties) XAML attached property.

### -param value
The property value to set.

## -remarks
This method is a utility method for the property system, and isn't used in most app scenarios. In most cases you set the [Grid.Row](/uwp/api/windows.ui.xaml.controls.grid#xaml-attached-properties) XAML attached property in XAML and won't need this method. For more info, see the [Grid.Row](/uwp/api/windows.ui.xaml.controls.grid#xaml-attached-properties) attached property.

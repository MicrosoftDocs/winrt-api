# Windows.UI.Xaml.Controls.Frame.CacheSize

## -description
Gets or sets the number of pages in the navigation history that can be cached for the frame.



## -xaml-syntax
```xaml
<Frame CacheSize="int" />
```


## -property-value
The number of pages that can be in the navigation history.

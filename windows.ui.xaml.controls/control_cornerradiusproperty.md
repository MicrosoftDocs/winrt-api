# Windows.UI.Xaml.Controls.Control.CornerRadiusProperty

## -description

Identifies the [CornerRadius](control_cornerradius.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.CornerRadiusProperty](/windows/winui/api/microsoft.ui.xaml.controls.control.cornerradiusproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the [CornerRadius](control_cornerradius.md) dependency property.

# Windows.UI.Xaml.Controls.SemanticZoomViewChangedEventArgs

## -description
Provides event data for the [ViewChangeStarted](semanticzoom_viewchangestarted.md) and [ViewChangeCompleted](semanticzoom_viewchangecompleted.md) events.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.SemanticZoomViewChangedEventArgs](/windows/winui/api/microsoft.ui.xaml.controls.semanticzoomviewchangedeventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

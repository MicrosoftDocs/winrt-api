# Windows.UI.Xaml.Controls.PasswordBox.PasteFromClipboard

## -description

Copies content from the Windows clipboard into the text control.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PasswordBox.PasteFromClipboard](/windows/winui/api/microsoft.ui.xaml.controls.passwordbox.pastefromclipboard) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

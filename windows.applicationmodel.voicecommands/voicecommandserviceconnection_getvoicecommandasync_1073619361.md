# Windows.ApplicationModel.VoiceCommands.VoiceCommandServiceConnection.GetVoiceCommandAsync

## -description
Retrieves the voice command that the user submitted to **Cortana** by voice or text.

## -returns
The [VoiceCommand](voicecommand.md).

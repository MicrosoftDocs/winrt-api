# PlayReadyITADataFormat

## -description
Specifies the format for the ITA serialized data.

## -enum-fields
### -field SerializedProperties:0
Returns the serialized [IPropertySet](../windows.foundation.collections/ipropertyset.md) as detailed in the remarks for the [PlayReadyITADataGenerator.GenerateData](playreadyitadatagenerator_generatedata.md) method.

### -field SerializedProperties_WithContentProtectionWrapper:1
Returns the entire serialized blob including the content protection system **GUID**, the stream count, and the next stream ID block.

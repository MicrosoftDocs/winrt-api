# Windows.Devices.HumanInterfaceDevice.HidDevice.UsagePage

## -description

Gets the usage page of the top-level collection.

## -property-value

The usage page.

## -remarks

The **UsagePage** of the top-level collection is retrieved from the HID report descriptor.

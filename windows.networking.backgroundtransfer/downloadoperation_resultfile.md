# Windows.Networking.BackgroundTransfer.DownloadOperation.ResultFile

## -description
Returns the [IStorageFile](../windows.storage/istoragefile.md) object provided by the caller when creating the [DownloadOperation](downloadoperation.md) object using [CreateDownload](backgrounddownloader_createdownload.md).

## -property-value
The returned file object.

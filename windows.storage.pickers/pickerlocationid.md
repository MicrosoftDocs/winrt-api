# PickerLocationId

## -description

Identifies the storage location that the file picker presents to the user.

## -enum-fields

### -field DocumentsLibrary:0

The **Documents** library.

### -field ComputerFolder:1

The **Computer** folder.

### -field Desktop:2

The Windows desktop.

### -field Downloads:3

The **Downloads** folder.

### -field HomeGroup:4

The HomeGroup.

### -field MusicLibrary:5

The **Music** library.

### -field PicturesLibrary:6

The **Pictures** library.

### -field VideosLibrary:7

The **Videos** library.

### -field Objects3D:8

The **Objects** library.

### -field Unspecified:9

An unspecified location.

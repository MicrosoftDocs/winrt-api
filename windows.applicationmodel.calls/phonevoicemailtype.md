# PhoneVoicemailType

## -description
The type of voice mail associated with a [PhoneLine](phoneline.md) instance.

## -enum-fields
### -field None:0
The phone line has no voice mail.

### -field Traditional:1
The phone line has traditional voice mail.

### -field Visual:2
The phone line has visual voice mail.

# FormatEffect

## -description
Defines values that indicate the state of a character or paragraph formatting property.



## -enum-fields
### -field Off:0
Turns off the property.

### -field On:1
Turns on the property.

### -field Toggle:2
Toggles the current setting.

### -field Undefined:3
No change.

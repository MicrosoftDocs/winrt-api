# Windows.UI.Composition.Scenes.SceneAlphaMode

## -description

Defines constants that specify the alpha mode for a scene.

## -enum-fields

### -field AlphaTest:1

Alpha mode is AlphaTest.

### -field Blend:2

Alpha mode is Blend.

### -field Opaque:0

Alpha mode is Opaque.

# Windows.UI.Xaml.Automation.Peers.MapControlAutomationPeer.HorizontalScrollPercent

## -description
Gets a [Double](/dotnet/api/system.double?view=dotnet-uwp-10.0&preserve-view=true) value indicating the percentage of how far the map has been scrolled horizontally.

## -property-value
A [Double](/dotnet/api/system.double?view=dotnet-uwp-10.0&preserve-view=true) value indicating the percentage of how far the map has been scrolled horizontally.

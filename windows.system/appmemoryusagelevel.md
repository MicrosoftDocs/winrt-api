# AppMemoryUsageLevel

## -description
An enumeration containing 3 levels of app memory usage: low, medium, and high.

## -enum-fields
### -field Low:0
Low app memory usage.

### -field Medium:1
Medium app memory usage.

### -field High:2
High app memory usage.

### -field OverLimit:3
App memory usage is over the limit.

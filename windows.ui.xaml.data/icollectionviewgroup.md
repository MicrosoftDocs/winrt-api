# Windows.UI.Xaml.Data.ICollectionViewGroup

## -description
Represents any grouped items within a view.

## -remarks
[ICollectionViewGroup](icollectionviewgroup.md) is used like a view model to represent a group within a collection. When implementing a custom grouped data source, your data source should implement [ICollectionViewFactory](icollectionviewfactory.md). For more info, see [ICollectionViewFactory](icollectionviewfactory.md) and [ICollectionView](icollectionview.md).

# Windows.UI.Xaml.Data.ICustomProperty.GetIndexedValue

## -description
Gets the value at an index location, for cases where the custom property has indexer support.



## -parameters
### -param target
The owning instance.

### -param index
The index to get.

## -returns
The retrieved value at the index.

# AutomationFlowDirections

## -description
Defines the direction of flow.



## -enum-fields
### -field Default:0
Flow is in the default direction.

### -field RightToLeft:1
Flow is from right to left.

### -field BottomToTop:2
Flow is from bottom to top.

### -field Vertical:3
Flow is vertical.

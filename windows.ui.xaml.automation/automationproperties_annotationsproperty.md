# Windows.UI.Xaml.Automation.AutomationProperties.AnnotationsProperty

## -description
Identifies a dependency property that's used for a list of annotation objects in a document, such as comment, header, footer, and so on.

Equivalent WinUI property: [Microsoft.UI.Xaml.Automation.AutomationProperties.AnnotationsProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties.annotationsproperty).

## -property-value
The identifier for the **Annotations** attached property.

## -remarks
There's no public dependency property for **Annotations**. Use [GetAnnotations](automationproperties_getannotations_1232704323.md) to get the list of annotations.

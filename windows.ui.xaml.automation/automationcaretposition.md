# AutomationCaretPosition

## -description
Defines the caret position.



## -enum-fields
### -field Unknown:0
Caret position is unknown.

### -field EndOfLine:1
Caret is at the end of the line.

### -field BeginningOfLine:2
Caret is at the beginning of the line.

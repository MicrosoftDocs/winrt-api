# Windows.UI.Xaml.Automation.AutomationProperties.SetCulture

## -description
Sets the value of the [CultureProperty](automationproperties_cultureproperty.md) attached property for the specified [DependencyObject](/uwp/api/windows.ui.xaml.dependencyobject).  

Equivalent WinUI method: [Microsoft.UI.Xaml.Automation.AutomationProperties.SetCulture](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties.setculture).

## -parameters

### -param element
The [DependencyObject](/uwp/api/windows.ui.xaml.dependencyobject) for which to set the culture property.  

### -param value
The value of the culture property to set for the specified *element*.

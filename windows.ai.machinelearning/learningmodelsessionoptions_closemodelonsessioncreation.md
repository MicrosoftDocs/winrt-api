# Windows.AI.MachineLearning.LearningModelSessionOptions.CloseModelOnSessionCreation

## -description
Specifies whether a [LearningModelSession](learningmodelsession.md) should close the associated learning model when it is created.

## -property-value
Indicates whether the model should be closed upon session creation.

# Windows.Storage.Search.IStorageFolderQueryOperations.IsCommonFolderQuerySupported

## -description
Verifies whether this folder supports the specified folder query (returns true or false).

## -parameters
### -param query
The folder query to test.

## -returns
True if the folder supports the specified folder query; otherwise false.

## -remarks
Not all locations support all query options, for more detail refer to [IsCommonFileQuerySupported](istoragefolderqueryoperations_iscommonfilequerysupported.md).

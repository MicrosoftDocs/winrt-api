# Windows.Globalization.Calendar.NumeralSystem

## -description
Gets or sets the numbering system that is used when formatting components as strings.

## -property-value
A string that indicates the numeral system of the calendar, such as "Latn" for the Latin numeral system (0123456789) or "Arab" for the Arabic-Indic numeral system (٠١٢٣٤٥٦٧٨٩). See the supported [NumeralSystem values](/previous-versions/windows/apps/jj236471(v=win.10)).

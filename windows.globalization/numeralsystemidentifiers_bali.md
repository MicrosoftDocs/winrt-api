# Windows.Globalization.NumeralSystemIdentifiers.Bali

## -description
Gets the identifier for the Balinese decimal numeral system.

## -property-value
The string "Bali", which identifies the Balinese decimal numeral system. This system uses the following Unicode values:<table>
   <tr><th>Unicode code point</th><th>Corresponding Latin digit</th></tr>
   <tr><td>U+1B50</td><td>0</td></tr>
   <tr><td>U+1B51</td><td>1</td></tr>
   <tr><td>U+1B52</td><td>2</td></tr>
   <tr><td>U+1B53</td><td>3</td></tr>
   <tr><td>U+1B54</td><td>4</td></tr>
   <tr><td>U+1B55</td><td>5</td></tr>
   <tr><td>U+1B56</td><td>6</td></tr>
   <tr><td>U+1B57</td><td>7</td></tr>
   <tr><td>U+1B58</td><td>8</td></tr>
   <tr><td>U+1B59</td><td>9</td></tr>
</table>

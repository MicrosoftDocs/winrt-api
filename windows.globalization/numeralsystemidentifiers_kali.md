# Windows.Globalization.NumeralSystemIdentifiers.Kali

## -description
Gets the identifier for the Kayah Li decimal numeral system.

## -property-value
The string "Kali", which identifies the Kayah Li decimal numeral system. This system uses the following Unicode values:<table>
   <tr><th>Unicode code point</th><th>Corresponding Latin digit</th></tr>
   <tr><td>U+A900</td><td>0</td></tr>
   <tr><td>U+A901</td><td>1</td></tr>
   <tr><td>U+A902</td><td>2</td></tr>
   <tr><td>U+A903</td><td>3</td></tr>
   <tr><td>U+A904</td><td>4</td></tr>
   <tr><td>U+A905</td><td>5</td></tr>
   <tr><td>U+A906</td><td>6</td></tr>
   <tr><td>U+A907</td><td>7</td></tr>
   <tr><td>U+A908</td><td>8</td></tr>
   <tr><td>U+A909</td><td>9</td></tr>
</table>

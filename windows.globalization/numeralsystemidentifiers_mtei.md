# Windows.Globalization.NumeralSystemIdentifiers.Mtei

## -description
Gets the identifier for the Meetei Mayek decimal numeral system.

## -property-value
The string "Mtei", which identifies the Meetei Mayek decimal numeral system. This system uses the following Unicode values:<table>
   <tr><th>Unicode code point</th><th>Corresponding Latin digit</th></tr>
   <tr><td>U+ABF0</td><td>0</td></tr>
   <tr><td>U+ABF1</td><td>1</td></tr>
   <tr><td>U+ABF2</td><td>2</td></tr>
   <tr><td>U+ABF3</td><td>3</td></tr>
   <tr><td>U+ABF4</td><td>4</td></tr>
   <tr><td>U+ABF5</td><td>5</td></tr>
   <tr><td>U+ABF6</td><td>6</td></tr>
   <tr><td>U+ABF7</td><td>7</td></tr>
   <tr><td>U+ABF8</td><td>8</td></tr>
   <tr><td>U+ABF9</td><td>9</td></tr>
</table>

# Windows.UI.Core.ClosestInteractiveBoundsRequestedEventArgs.ClosestInteractiveBounds

## -description
> [!NOTE]
> Not intended for general use.

Gets or sets the bounding rectangle of the interactive element that is within the [SearchBounds](closestinteractiveboundsrequestedeventargs_searchbounds.md) and closest to the [PointerPosition](closestinteractiveboundsrequestedeventargs_pointerposition.md).

## -property-value
The bounding rectangle of the interactive element that is within the [SearchBounds](closestinteractiveboundsrequestedeventargs_searchbounds.md) and closest to the [PointerPosition](closestinteractiveboundsrequestedeventargs_pointerposition.md).

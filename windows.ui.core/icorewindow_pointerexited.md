# Windows.UI.Core.ICoreWindow.PointerExited

## -description
Specifies the event that occurs when the pointer moves outside the bounding box of the window.

## -remarks
For touch points this event will be raised before the [PointerReleased](icorewindow_pointerpressed.md) event.

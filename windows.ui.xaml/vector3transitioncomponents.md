# Windows.UI.Xaml.Vector3TransitionComponents

## -description
Defines constants that specify which axes to animate during the Vector3Transition animation. 



## -enum-fields
### -field X:1
Animate the X axis.

### -field Y:2
Animate the Y axis.

### -field Z:4
Animate the Z axis.

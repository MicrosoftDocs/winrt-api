# Windows.UI.Xaml.DragOperationDeferral

## -description

Enables you to drag-and-drop content with a target app asynchronously.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.DragOperationDeferral](/windows/winui/api/microsoft.ui.xaml.dragoperationdeferral) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

# Windows.Media.Audio.CreateAudioGraphResult.Status

## -description
Gets the result status of an operation to create an [AudioGraph](audiograph.md) with a call to [AudioGraph.CreateAsync](audiograph_createasync_145006550.md).

## -property-value
An enumeration value indicating audio graph creation status.

## -remarks
If the result status is **Success**, get the instance of the audio graph with the [Graph](createmediasourceaudioinputnoderesult_graph.md) property.

# Windows.Media.Audio.AudioDeviceInputNode.Reset

## -description
Resets the audio device input node.

## -remarks
Calling this method causes [DiscardQueuedFrames](../windows.media.effects/ibasicaudioeffect_discardqueuedframes.md) to be called on any audio effects applied to the [AudioDeviceInputNode](audiodeviceinputnode.md).

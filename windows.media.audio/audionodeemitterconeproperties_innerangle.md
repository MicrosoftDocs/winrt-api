# Windows.Media.Audio.AudioNodeEmitterConeProperties.InnerAngle

## -description
Gets the inner angle of an audio node emitter cone shape. This is the angle at which the audio is not attenuated at all.

## -property-value
The inner angle of a an audio node emitter cone shape.

## -remarks
Set the inner angle of an emitter cone shape by calling [AudioNodeEmitterShape.CreateCone](audionodeemittershape_createcone.md).

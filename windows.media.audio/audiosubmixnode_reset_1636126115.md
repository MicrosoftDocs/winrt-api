# Windows.Media.Audio.AudioSubmixNode.Reset

## -description
Resets the audio submix node.

## -remarks
Calling this method causes [DiscardQueuedFrames](../windows.media.effects/ibasicaudioeffect_discardqueuedframes.md) to be called on any audio effects applied to the [AudioSubmixNode](audiosubmixnode.md).

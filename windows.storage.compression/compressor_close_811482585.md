# Windows.Storage.Compression.Compressor.Close

## -description
Closes a Compressor object and synchronously discards any information in buffers. This will close the underlying stream as well unless the [Compressor.Detach](compressor_detachstream.md) method has been used to detach the stream from the object. Subsequent calls on a closed object, except for , will fail.

# Windows.Services.Maps.MapRouteFinder.GetWalkingRouteAsync

## -description
Gets a walking route using the specified start and end coordinates.

## -parameters
### -param startPoint
The starting point of the walking route.

### -param endPoint
The ending point of the walking route.

## -returns
When this method completes successfully, it returns a walking route contained in the [MapRouteFinderResult](maproutefinderresult.md).

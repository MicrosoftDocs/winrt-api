# Windows.Services.Maps.MapRouteLeg.Maneuvers

## -description
Gets the list of maneuvers associated with the route leg.

## -property-value
The list of maneuvers associated with the route leg. This property returns a collection of [MapRouteManeuver](maproutemaneuver.md) objects.

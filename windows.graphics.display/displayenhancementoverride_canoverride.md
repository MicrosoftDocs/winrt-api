# Windows.Graphics.Display.DisplayEnhancementOverride.CanOverride



## -description

Gets a value that describes whether the current display settings can be overridden.

## -property-value

**True** if the [DisplayEnhancementOverride](displayenhancementoverride.md) object can successfully override the system's brightness and/or color with its current configuration at the moment that [CanOverride](displayenhancementoverride_canoverride.md)is queried. This can be used to test if override control can be obtained without having to call [RequestOverride](displayenhancementoverride_requestoverride_158486181.md).

# ForegroundText

## -description
Specifies the color of the tile's foreground text.

> [!NOTE]
> This value is ignored on Windows Phone 8.1.

## -enum-fields
### -field Dark:0
A Windows-specified default dark text color.

### -field Light:1
A Windows-specified default light text color.

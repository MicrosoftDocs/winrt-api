# Windows.Web.Http.HttpRequestMessage.RequestUri

## -description
Gets or sets the [Uri](../windows.foundation/uri.md) used for the [HttpRequestMessage](httprequestmessage.md) object.

## -property-value
The [Uri](../windows.foundation/uri.md) used for the HTTP request.

## -remarks
The *value* of the [RequestUri](httprequestmessage_requesturi.md) must not be a relative [Uri](../windows.foundation/uri.md).

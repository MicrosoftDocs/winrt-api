# Windows.Web.Http.HttpStreamContent.HttpStreamContent

## -description
Initializes a new instance of the [HttpStreamContent](httpstreamcontent.md) class with the specified content.

## -parameters
### -param content
The content used to initialize the [HttpStreamContent](httpstreamcontent.md).

## -exceptions
### E_INVALIDARG

The *content* parameter is **null** (**Nothing** in Visual Basic).

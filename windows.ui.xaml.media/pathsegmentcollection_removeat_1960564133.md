# Windows.UI.Xaml.Media.PathSegmentCollection.RemoveAt

## -description
Removes the item at the specified index.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.PathSegmentCollection.RemoveAt](/windows/winui/api/microsoft.ui.xaml.media.pathsegmentcollection.removeat) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index
The index position of the item to remove.

# Windows.UI.Xaml.Media.TransformGroup

## -description
Represents a composite [Transform](transform.md) composed of other [Transform](transform.md) objects.



## -xaml-syntax
```xaml
<TransformGroup>
  oneOrMoreTransforms
</TransformGroup>
```

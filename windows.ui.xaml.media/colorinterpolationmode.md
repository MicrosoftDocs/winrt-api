# ColorInterpolationMode

## -description
Determines how the colors in a gradient are interpolated.

Equivalent **WinUI 2 API for UWP**: [ColorInterpolationMode](/windows/winui/api/microsoft.ui.xaml.media.colorinterpolationmode) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

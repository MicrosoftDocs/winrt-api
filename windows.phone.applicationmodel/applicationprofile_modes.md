# Windows.Phone.ApplicationModel.ApplicationProfile.Modes

## -description
Gets a value that indicates the mode that an app is running in.

## -property-value
A value that indicates the mode that an app is running in.

## -remarks
Use the value of this property to determine if a Windows Phone app is currently running in Kid's Corner. A value of Default indicates that the app is running in normal mode. A value of Alternate indicates that the app is running in Kid's Corner mode. You can use this feature to disable features of your app while it is running in Kid's Corner. For more info, see [Kid's Corner for Windows Phone](/previous-versions/windows/apps/dn168931(v=vs.105))

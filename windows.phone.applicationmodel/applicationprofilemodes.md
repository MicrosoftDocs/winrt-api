# ApplicationProfileModes

## -description
Indicates the mode that an app is running in.

## -enum-fields
### -field Default:0
The app is running in default mode.

### -field Alternate:1
The app is running in alternate mode.


## -remarks
The Alternate value indicates that a Windows Phone app is running in Kid's Corner mode. For more info, see [Kid's Corner for Windows Phone](/previous-versions/windows/apps/dn168931(v=vs.105))

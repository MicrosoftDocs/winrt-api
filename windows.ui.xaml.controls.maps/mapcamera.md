# Windows.UI.Xaml.Controls.Maps.MapCamera

## -description
Represents the position of a camera to define the view of a map.

## -remarks
The [MapCamera](mapcamera.md) class uses a position to define the view of a map in the same way that the position of a real camera determines it's view of the earth.

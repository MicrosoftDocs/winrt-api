# Windows.UI.Xaml.Controls.Maps.MapPolyline.StrokeThickness

## -description
Gets or sets the width of the line used to draw the [MapPolyline](mappolyline.md), in logical pixels.

## -property-value
The width of the line used to draw the [MapPolyline](mappolyline.md), in logical pixels.

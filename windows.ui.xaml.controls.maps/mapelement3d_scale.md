# Windows.UI.Xaml.Controls.Maps.MapElement3D.Scale

## -description
The scale to apply to the 3D map element.

## -property-value
The scale to apply to the 3D map element. A value of Vector3 (1.0,1.0,1.0) is equivalent to 100%.

# MapLoadingStatus

## -description
Specifies the [LoadingStatus](mapcontrol_loadingstatus.md) of the [MapControl](mapcontrol.md).

## -enum-fields
### -field Loading:0
The map is loading.

### -field Loaded:1
The map is loaded.

### -field DataUnavailable:2
Map data is unavailable.

# Windows.UI.Xaml.Controls.Maps.MapControl.TryZoomOutAsync

## -description
Starts an asynchronous operation to zoom out, decreasing the map's [ZoomLevel](mapcontrol_zoomlevel.md) by one.

## -returns
Contains the results of the operation: **true** if the operation completed successfully; otherwise, **false**.

## -remarks
See the remarks section of the [TryZoomInAsync](mapcontrol_tryzoominasync.md) method.

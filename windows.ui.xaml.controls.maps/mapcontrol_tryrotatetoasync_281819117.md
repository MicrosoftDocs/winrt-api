# Windows.UI.Xaml.Controls.Maps.MapControl.TryRotateToAsync

## -description
Starts an asynchronous operation to rotate the map's camera to a specific orientation.

## -parameters
### -param angleInDegrees
The final [Heading](mapcamera_heading.md) of the map's camera, in degrees.

## -returns
Contains the results of the operation: **true** if the operation completed successfully; otherwise, **false**.

## -remarks
See the remarks section of the [TryRotateAsync](mapcontrol_tryrotateasync.md) method.

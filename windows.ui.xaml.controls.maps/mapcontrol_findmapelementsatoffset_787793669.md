# Windows.UI.Xaml.Controls.Maps.MapControl.FindMapElementsAtOffset

## -description
Retrieves the collection of [MapElement](mapcontrol_mapelements.md) objects at the specified point on the map.

## -parameters

### -param offset
The point on the map from which to retrieve the collection of [MapElement](mapelement.md) objects.

## -returns
The collection of [MapElement](mapcontrol_mapelements.md) objects at the specified point on the map.

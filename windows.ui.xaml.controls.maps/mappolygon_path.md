# Windows.UI.Xaml.Controls.Maps.MapPolygon.Path

## -description
Gets or sets the collection of coordinates that define the [MapPolygon](mappolygon.md) shape.

> [!NOTE]
> In WindowsÂ 10, version 1511, we recommend using the [Paths](mappolygon_paths.md) property to create simple and complex polygons. Do not use both properties ([Paths](mappolygon_paths.md) and Path) at the same time.

## -property-value
The collection of coordinates that define the [MapPolygon](mappolygon.md) shape.

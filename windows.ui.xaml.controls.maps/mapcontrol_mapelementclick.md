# Windows.UI.Xaml.Controls.Maps.MapControl.MapElementClick

## -description
Occurs when the user taps or clicks a [MapElement](mapelement.md) on the [MapControl](mapcontrol.md).

## -remarks
Touch, mouse, and pen/stylus interactions are received, processed, and managed as pointers. Any of these devices and their interactions can produce a [MapElementClick](mapcontrol_mapelementclick.md) event. An instance of [MapElementClickEventArgs](mapelementclickeventargs.md) provides data for this event.

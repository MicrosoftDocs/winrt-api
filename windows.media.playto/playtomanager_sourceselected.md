# Windows.Media.PlayTo.PlayToManager.SourceSelected

## -description
Occurs when a Play To source element has been selected.

## -remarks

In a Play Toâ€“ enabled app, the user selects the **Devices** charm (or the Play To UI) to stream audio, video, or images from the app to a target device. The SourceSelected event is fired after the user has selected a target device. If the user exits the **Devices** charm or the Play To UI without selecting a target device, the SourceSelected event isn't fired.

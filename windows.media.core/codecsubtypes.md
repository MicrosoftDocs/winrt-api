# Windows.Media.Core.CodecSubtypes

## -description
Provides properties for most supported media subtypes that return the string representation of the GUID for the subtype. Use these property values to filter a query performed with [CodecQuery.FindAllAsync](codecquery_findallasync_1027997915.md). You can also compare the values returned by these properties to the values in the [Subtypes](./codecinfo_subtypes.md) collection of a [CodecInfo](codecinfo.md) object to determine if a codec supports a particular media subtype.

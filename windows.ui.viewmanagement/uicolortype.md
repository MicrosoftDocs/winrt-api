# UIColorType

## -description
Defines constants that specify known system color values.

## -enum-fields
### -field Background:0
The background color.

### -field Foreground:1
The foreground color.

### -field AccentDark3:2
The darkest accent color.

### -field AccentDark2:3
The darker accent color.

### -field AccentDark1:4
The dark accent color.

### -field Accent:5
The accent color.

### -field AccentLight1:6
The light accent color.

### -field AccentLight2:7
The lighter accent color.

### -field AccentLight3:8
The lightest accent color.

### -field Complement:9
Not supported. Do not use.


## -remarks
See [UISettings.GetColorValue](uisettings_getcolorvalue.md).

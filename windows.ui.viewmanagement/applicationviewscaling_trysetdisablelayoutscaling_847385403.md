# Windows.UI.ViewManagement.ApplicationViewScaling.TrySetDisableLayoutScaling

## -description
Attempts to set the [DisableLayoutScaling](applicationviewscaling_disablelayoutscaling.md) property to the specified value.

## -parameters
### -param disableLayoutScaling
**true** to disable layout scaling; **false** to enable it.

## -returns
**true** if the property was set successfully; otherwise, **false**.

# Windows.System.RemoteSystems.RemoteSystemSessionValueSetReceivedEventArgs.Sender

## -description
Gets the remote session participant that sent the message to the channel and caused this [ValueSetReceived](remotesystemsessionmessagechannel_valuesetreceived.md) event to be raised.

## -property-value
The [RemoteSystemSessionParticipant](RemoteSystemSessionParticipant.md) object corresponding to the sender participant.

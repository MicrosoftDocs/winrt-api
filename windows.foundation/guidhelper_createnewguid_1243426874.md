# Windows.Foundation.GuidHelper.CreateNewGuid

## -description
Creates a new, unique **Guid**.

## -returns
A new, unique **Guid**.

## -remarks
In JavaScript, **Guid** is projected as a string.

## -see-also

## -examples
```javascript
var newGuid = Windows.Foundation.GuidHelper.createNewGuid();
```

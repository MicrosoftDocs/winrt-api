## -description

## -examples
This example shows how to use a Polyline to create a closed triangle.

```xaml
<Canvas>  
  <Polyline Points="50,25 0,100 100,100 50,25"
    Stroke="Blue" StrokeThickness="10"
    Canvas.Left="75" Canvas.Top="50" />
</Canvas>
```

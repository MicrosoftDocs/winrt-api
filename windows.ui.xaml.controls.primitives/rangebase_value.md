# Windows.UI.Xaml.Controls.Primitives.RangeBase.Value

## -description
Gets or sets the current setting of the range control, which may be coerced.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.RangeBase.Value](/windows/winui/api/microsoft.ui.xaml.controls.primitives.rangebase.value) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

# Windows.UI.Xaml.Controls.Primitives.Selector.SelectedValueProperty

## -description
Gets the identifier for the [SelectedValue](selector_selectedvalue.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.Selector.SelectedValueProperty](/windows/winui/api/microsoft.ui.xaml.controls.primitives.selector.selectedvalueproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [SelectedValue](selector_selectedvalue.md) dependency property.

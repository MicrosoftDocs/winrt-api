# Windows.Graphics.Printing3D.Printing3DBaseMaterial

## -description
Represents the base material used for manufacturing certain objects in the 3D model.

## -remarks
The [Printing3DBaseMaterial](printing3dbasematerial.md) class corresponds to the `<base>` element in the 3D Manufacturing Format (3MF) specification. For more info, see the [3MF Specification](http://go.microsoft.com/fwlink/p/?LinkId=616252).

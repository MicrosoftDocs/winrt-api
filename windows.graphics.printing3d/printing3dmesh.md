# Windows.Graphics.Printing3D.Printing3DMesh

## -description
Represents a mesh in the 3D model.

## -remarks
The Printing3DMesh class corresponds to the `<mesh>` element in the 3D Manufacturing Format (3MF) specification. For more info, see the [3MF Specification](http://3mf.io/what-is-3mf/3mf-specification/).

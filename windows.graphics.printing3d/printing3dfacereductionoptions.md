# Windows.Graphics.Printing3D.Printing3DFaceReductionOptions

## -description
Provides additional information for the [TryReduceFacesAsync](printing3dmodel_tryreducefacesasync_1526716101.md) method in the [Printing3DModel](printing3dmodel.md) class

## -remarks
This API is designed for use with 3D Manufacturing Format (3MF) packages. For more info about 3MF, see the [3MF Specification](http://3mf.io/what-is-3mf/3mf-specification/).

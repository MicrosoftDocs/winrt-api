# Windows.Graphics.Printing3D.Printing3DBaseMaterialGroup

## -description
Represents a group of base materials used in the 3D model.

## -remarks
The Printing3DBaseMaterialGroup class corresponds to the `<basematerials>` element in the 3D Manufacturing Format (3MF) specification. For more info, see the [3MF Specification](http://3mf.io/what-is-3mf/3mf-specification/).

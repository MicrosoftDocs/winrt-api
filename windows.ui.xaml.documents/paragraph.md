# Windows.UI.Xaml.Documents.Paragraph

## -description
Provides a block-level content element that is used to group content into a paragraph.



## -xaml-syntax
```xaml
<Paragraph .../>
-or-
<Paragraph ...>
  oneOrMoreInlines
</Paragraph>
-or-
<Paragraph ...>
  mixedContent
</Paragraph>
```

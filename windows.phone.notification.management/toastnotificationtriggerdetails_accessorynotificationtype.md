# Windows.Phone.Notification.Management.ToastNotificationTriggerDetails.AccessoryNotificationType

## -description
An [AccessoryNotificationType](accessorynotificationtype.md) that specifies the type of notification received. For toast notifications, this value is AccessoryNotificationType.Toast.

## -property-value
AccessoryNotificationType.Toast

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.

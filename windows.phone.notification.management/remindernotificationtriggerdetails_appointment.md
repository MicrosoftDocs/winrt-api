# Windows.Phone.Notification.Management.ReminderNotificationTriggerDetails.Appointment

## -description
An [Appointment](../windows.applicationmodel.appointments/appointment.md) object that contains additional details about the reminder.

## -property-value
For additional information, see [Appointment](../windows.applicationmodel.appointments/appointment.md).

## -remarks
This property is used only for time-based reminders, such as Calendar events. For other reminder types, this value is null.

Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.

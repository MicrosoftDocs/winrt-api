# Windows.UI.Input.Inking.InkPresenterRuler.Length

## -description
Gets or sets the length of the [InkPresenterRuler](inkpresenterruler.md).

## -property-value

The length in device-independent pixel (DIP).

+ Default: 1600 device-independent pixel (DIP)
+ Minimum: 300 device-independent pixel (DIP)

# Windows.UI.Input.Inking.InkStrokeContainer.GetStrokeById

## -description
Retrieves the single InkStroke with the specified ID.

## -parameters
### -param id
The Global Unique Identifier (GUID) used to identify a single ink stroke.

## -returns
The single ink stroke.

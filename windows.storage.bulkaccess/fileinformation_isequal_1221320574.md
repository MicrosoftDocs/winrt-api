# Windows.Storage.BulkAccess.FileInformation.IsEqual

## -description
Indicates whether the current file is the same as the specified storage item.

## -parameters
### -param item
The [IStorageItem](../windows.storage/istorageitem.md) object that represents a storage item to compare against.

## -returns
Returns **true** if the current storage item is the same as the specified storage item; otherwise **false**.

# Windows.UI.Composition.CompositionPropertySet.InsertColor

## -description
Inserts a [Color](../windows.ui/color.md) key-value pair.



## -parameters
### -param propertyName
The key associated with the value. This key can be used to retrieve the value.

### -param value
The value to insert.

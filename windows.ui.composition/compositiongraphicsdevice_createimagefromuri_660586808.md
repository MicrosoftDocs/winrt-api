# Windows.UI.Composition.CompositionGraphicsDevice.CreateImageFromUri(Windows.Foundation.Uri)

## -description

Creates a CompositionImage from the specified Uri.

## -parameters

The Uri to create the image from.

## -returns

The created CompositionImage.

## -remarks

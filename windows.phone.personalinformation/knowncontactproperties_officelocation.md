# Windows.Phone.PersonalInformation.KnownContactProperties.OfficeLocation

## -description
Gets the **OfficeLocation** property name.

## -property-value
The **OfficeLocation** property name.

## -remarks
The value returned for the **OfficeLocation** key in a contact **Properties** collection is a string.

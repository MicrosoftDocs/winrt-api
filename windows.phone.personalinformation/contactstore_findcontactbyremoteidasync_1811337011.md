# Windows.Phone.PersonalInformation.ContactStore.FindContactByRemoteIdAsync

## -description
Retrieves the contact with the specified remote ID from the contact store.

## -parameters
### -param id
The application-defined remote ID of the contact to retrieve.

## -returns
When this method completes, it returns a [StoredContact](storedcontact.md) object. If you use [Asynchronous programming](https://msdn.microsoft.com/library/23fe28f1-89c5-4a17-a732-a722648f9c5e), the result type is [StoredContact](storedcontact.md).

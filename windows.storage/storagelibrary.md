# Windows.Storage.StorageLibrary

## -description

Lets you add and remove folders from a library like Music or Videos and lets you get a list of folders included in the library.

## -remarks

You can only access libraries that you have declared capabilities for in your app manifest. To learn more about capabilities see, [App capability declarations](http://msdn.microsoft.com/library/25b18ba5-e584-4537-9f19-bb2c8c52dfe1).

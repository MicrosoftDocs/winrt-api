# Windows.Storage.CachedFileManager.DeferUpdates

## -description
Lets apps defer real-time updates for a specified file.

## -parameters
### -param file
The file to defer updates for.

## -remarks
Typically, Windows implicitly initiates updates for files that are provided by other apps when those files change.Â  However, you can control when updates are initiated by calling [DeferUpdates](cachedfilemanager_deferupdates.md).

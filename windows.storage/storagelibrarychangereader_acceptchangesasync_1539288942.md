# Windows.Storage.StorageLibraryChangeReader.AcceptChangesAsync

## -description
Tells the system that all of the changes described in the [StorageLibraryChange](storagelibrarychange.md) objects returned by the call to [ReadBatchAsync](storagelibrarychangereader_readbatchasync.md) have been addressed by the app.

## -returns
An object that is used to manage the asynchronous accept change operation.

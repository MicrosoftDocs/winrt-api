# Windows.Storage.IStorageFolder.GetItemAsync

## -description
Gets the specified item from the [IStorageFolder](istoragefolder.md).

## -parameters
### -param name
The name of the item to retrieve.

## -returns
When this method completes successfully, it returns the file or folder (type [IStorageItem](istorageitem.md)).

# Windows.Networking.NetworkOperators.MobileBroadbandAccount.GetConnectionProfiles

## -description
Retrieves an array of [ConnectionProfile](../windows.networking.connectivity/connectionprofile.md) objects that represent connections associated with the Mobile Broadband account.

## -returns
An array of [ConnectionProfile](../windows.networking.connectivity/connectionprofile.md) objects.

# Windows.Networking.NetworkOperators.HotspotAuthenticationContext.RedirectMessageXml

## -description
Gets the XML blob of the Wireless Internet Service Provider roaming (WISPr) redirect message of the hotspot.

> [!NOTE]
> This functionality is only available to mobile operator apps and UWP apps given privileged access by mobile network operators.



> If you want to use this API and publish your app to the Store, you will need special approval. For more information, see the **Special and restricted capabilities** section under [App capability declarations](/windows/uwp/packaging/app-capability-declarations). 

## -property-value
The WISPr XML blob from the hotspot redirect message.

# Windows.Security.Cryptography.Core.CryptographicHash.GetValueAndReset

## -description
Gets hashed data from the [CryptographicHash](cryptographichash.md) object and resets the object.

## -returns
Hashed data.

## -remarks
Call the [Append](cryptographichash_append.md) method to add data to the [CryptographicHash](cryptographichash.md) object.

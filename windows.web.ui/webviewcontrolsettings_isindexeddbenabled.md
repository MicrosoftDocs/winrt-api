# Windows.Web.UI.WebViewControlSettings.IsIndexedDBEnabled

## -description

Gets or sets a value that indicates whether the use of IndexedDB is allowed.

## -property-value

**true** if IndexedDB is allowed; otherwise, **false**. The default is **true**.

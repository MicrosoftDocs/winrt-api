# Windows.Devices.SerialCommunication.SerialDevice.GetDeviceSelector

## -description
Gets an Advanced Query Syntax (AQS) string that the app can pass to [DeviceInformation.FindAllAsync](../windows.devices.enumeration/deviceinformation_findallasync.md) in order to find all serial devices on the system.

## -returns
String formatted as an AQS query.

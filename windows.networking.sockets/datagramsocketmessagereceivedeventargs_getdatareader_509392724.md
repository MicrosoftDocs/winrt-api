# Windows.Networking.Sockets.DatagramSocketMessageReceivedEventArgs.GetDataReader

## -description
Gets a [DataReader](../windows.storage.streams/datareader.md) object to read incoming data received from the remote network destination on a [DatagramSocket](datagramsocket.md) object.

## -returns
A [DataReader](../windows.storage.streams/datareader.md) object to read incoming data received from the remote network destination on a [DatagramSocket](datagramsocket.md) object.

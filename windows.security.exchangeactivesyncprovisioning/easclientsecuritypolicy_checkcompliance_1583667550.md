# Windows.Security.ExchangeActiveSyncProvisioning.EasClientSecurityPolicy.CheckCompliance

## -description
> [!NOTE]
> [EasClientSecurityPolicy](easclientsecuritypolicy.md) may be unavailable for releases after WindowsÂ 10.

Evaluates the EAS policies.[CheckCompliance](easclientsecuritypolicy_checkcompliance.md) is called by a Windows Store app for mail any time the app wants to evaluate whether the local computer is compliant to the given EAS policies. Because this call doesn't involve any UI interactions, it is a synchronous call.

## -returns
Returns the results of the compliance check, in synchronous mode.

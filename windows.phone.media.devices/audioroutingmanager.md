# Windows.Phone.Media.Devices.AudioRoutingManager

## -description
Provides functionality that lets Voice over IP applications route audio to endpoints other than the speakerphone.

## -remarks
The APIs in the [Windows.Phone.Media.Devices](windows_phone_media_devices.md) namespace require the ID_CAP_AUDIOROUTING capability.

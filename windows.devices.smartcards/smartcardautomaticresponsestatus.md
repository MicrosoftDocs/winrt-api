# SmartCardAutomaticResponseStatus

## -description
Defines the status of the smart card readers automatic response.

## -enum-fields
### -field None:0
No status provided.

### -field Success:1
Read was a success.

### -field UnknownError:2
Read failed with an unidentified error.

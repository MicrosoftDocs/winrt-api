# SmartCardCryptogramMaterialPackageFormat

## -description
Defines the cryptogram material package format.

## -enum-fields
### -field None:0
The material does not have a format.

### -field JweRsaPki:1
The material package is JSON Web Encrypted (JWE) by RSA.

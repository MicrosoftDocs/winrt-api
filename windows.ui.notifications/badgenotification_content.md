# Windows.UI.Notifications.BadgeNotification.Content

## -description
Gets the XML that defines the value or glyph used as the tile's badge.

## -property-value
The object that contains the XML.

## -remarks
The retrieved elements and their attributes are manipulated through Document Object Model (DOM) manipulation functions to customize the badge content. See [badge schema](https://docs.microsoft.com/uwp/schemas/tiles/badgeschema/schema-root)for an explanation of badge elements and attributes.

# Windows.UI.Notifications.TileUpdateManager.CreateTileUpdaterForApplication

## -description
Creates and initializes a new instance of the [TileUpdater](tileupdater.md), which lets you change the appearance of the calling app's tile.

## -returns
The object you will use to send changes to the app's tile.

# NetworkUsageStates

## -description
Defines the desired state of the connection profile for which usage data is returned by the method [GetNetworkUsageAsync](connectionprofile_getnetworkusageasync_665790436.md).

## -struct-fields

### -field Roaming
Defines the desired roaming state of the network connection.
    

### -field Shared
Defines the desired sharing state of the network connection.
    

## -remarks
The Shared NetworkUsageState corresponds to when a network connection is made available to provide internet access for other devices. For an example of how NetworkUsageStates is used, see [How to retrieve connection usage data for a specific period of time](/previous-versions/windows/apps/hh465162(v=win.10)).

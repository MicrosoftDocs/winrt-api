# RoamingStates

## -description
Defines the roaming states.

## -enum-fields
### -field None:0
No roaming information.

### -field NotRoaming:1
The connection is not currently roaming.

### -field Roaming:2
The connection is currently roaming.


## -remarks
For examples of how these values are used in cost-based connection scenarios, see [Quickstart: Managing metered network cost constraints](/previous-versions/windows/apps/hh750310(v=win.10)).

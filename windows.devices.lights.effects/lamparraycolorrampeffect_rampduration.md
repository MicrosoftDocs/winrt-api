# Windows.Devices.Lights.Effects.LampArrayColorRampEffect.RampDuration

## -description
Gets or sets the time spent ramping-up to peak intensity from the previously set color.
## -property-value
Default value is 0.
## -remarks
Once the effect has been [Appended](lamparrayeffectplaylist_append_292269384.md) to a [LampArrayEffectPlaylist](lamparrayeffectplaylist.md), the value is locked and is not possible to set the value.
